## Milestone Goal

> Transform HelixAI from a stateless generate-and-download tool into a persistent platform where users log in with Google, maintain a sidebar of past conversations, pick up where they left off, and re-download their most recent preset per chat. Anonymous usage remains fully functional; login unlocks history.

## Critical Gap: UXP-01 / MISS-01

### Sign-in Banner OAuth CTA Non-Functional

**File:** `src/app/page.tsx` line 1499
**Current code:**
```jsx
onClick={() => window.dispatchEvent(new Event('helixtones:before-signin'))}
```

**What happens:** Dispatches `helixtones:before-signin` → `page.tsx` useEffect handler serializes chat state to sessionStorage. **Nothing else happens.** OAuth redirect never fires.

**What should happen:** After state serialization, call Supabase `linkIdentity()` (anonymous user) or `signInWithOAuth()` (no session) to redirect to Google.

**Fix:** Replicate `AuthButton.handleSignIn()` logic in the banner onClick — dispatch the event, await state serialization tick, then call the appropriate Supabase OAuth method using `createSupabaseBrowserClient()`.
