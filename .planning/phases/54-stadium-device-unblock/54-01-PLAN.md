**Fix 3 — Fixed Grid Positions (stadium-builder.ts):**
The amp block must land at position b05 and cab block at b06, matching real .hsp file layout. Currently all effect blocks use sequential `flowPos` starting at 1.

Replace the sequential `flowPos` counter approach (lines ~240-257) with a fixed grid allocation. The logic:
- Effects BEFORE the amp get positions b01, b02, b03, b04 (up to 4 pre-amp effects)
- Amp is ALWAYS at b05
- Cab is ALWAYS at b06
- Effects AFTER the cab get positions b07, b08, b09, b10, b11, b12 (up to 6 post-amp effects)

Implementation approach — replace the existing loop with two-pass logic:

```typescript
// Fixed grid positions per real .hsp layout
const AMP_SLOT = 5;
const CAB_SLOT = 6;

// Separate blocks into pre-amp, amp, cab, and post-amp
const preAmpBlocks: Array<[number, typeof activeBlocks[0]]> = [];
const postAmpBlocks: Array<[number, typeof activeBlocks[0]]> = [];
let ampBlock: [number, typeof activeBlocks[0]] | null = null;
let cabBlock: [number, typeof activeBlocks[0]] | null = null;

for (const [originalIndex, block] of activeBlocks.entries()) {
  if (block.type === "amp") {
    ampBlock = [originalIndex, block];
  } else if (block.type === "cab") {
    cabBlock = [originalIndex, block];
  } else if (ampBlock === null) {
    preAmpBlocks.push([originalIndex, block]);
  } else {
    postAmpBlocks.push([originalIndex, block]);
  }
}

// Place pre-amp effects at b01-b04
let prePos = 1;
for (const [originalIndex, block] of preAmpBlocks) {
  const blockKey = `b${String(prePos).padStart(2, "0")}`;
  flow0[blockKey] = buildFlowBlock(block, prePos, spec, originalIndex, blockKeyMap);
  prePos++;
}

// Place amp at b05
if (ampBlock) {
  const [originalIndex, block] = ampBlock;
  const blockKey = `b${String(AMP_SLOT).padStart(2, "0")}`;
  ampFlowPosition = AMP_SLOT;
  flow0[blockKey] = buildFlowBlock(block, AMP_SLOT, spec, originalIndex, blockKeyMap);
}

// Place cab at b06
if (cabBlock) {
  const [originalIndex, block] = cabBlock;
  const blockKey = `b${String(CAB_SLOT).padStart(2, "0")}`;
  cabFlowPosition = CAB_SLOT;
  flow0[blockKey] = buildFlowBlock(block, CAB_SLOT, spec, originalIndex, blockKeyMap);
}

// Place post-amp effects at b07-b12
let postPos = 7;
for (const [originalIndex, block] of postAmpBlocks) {
  const blockKey = `b${String(postPos).padStart(2, "0")}`;
  flow0[blockKey] = buildFlowBlock(block, postPos, spec, originalIndex, blockKeyMap);
  postPos++;
}
```

Preserve the existing `ampFlowPosition` and `cabFlowPosition` variable assignments that are used later for routing. Keep the `blockKeyMap` tracking. Keep `b00` (input) and `b13` (output) assignments unchanged — those are already correct.
