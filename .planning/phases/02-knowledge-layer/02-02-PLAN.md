<objective>
Create `param-engine.ts` -- the parameter resolution module that fills every block's parameters with expert-consensus values based on amp category, topology, and model defaults.

Purpose: This module eliminates the #1 cause of mediocre presets: AI-generated numeric parameter values. Instead, every Drive, Master, SAG, LowCut, HighCut, Mic, EQ, boost, and gate parameter comes from deterministic lookup tables derived from expert analysis of 250+ professional presets. Parameters are layered: model defaults (from AMP_MODELS) as base, category overrides on top, topology adjustments for high-gain mids.

Output: `src/lib/helix/param-engine.ts` with exported `resolveParameters()` function.
</objective>

```typescript
// 1. Start with the model's own defaults (per-model tuned in Phase 1)

  params.Mid = TOPOLOGY_MID[ampModel.topology] ?? params.Mid;
}
```
