# Phase 34 Plan — Stadium Chain Rules + Validation

**Phase:** 34
**Plan:** 01
**Requirements:** STAD-04

## Objective

Update `chain-rules.ts` and `validate.ts` to handle the Stadium device:

- `assembleSignalChain(intent, "helix_stadium")`: resolve amp from STADIUM_AMPS, use Stadium 7-band Parametric EQ as mandatory EQ, cap at STADIUM_MAX_BLOCKS_PER_PATH (12), single-path (Path 1A) only
- `validatePresetSpec(spec, "helix_stadium")`: include Agoura model IDs in valid set, enforce 12-block limit per path, enforce 8 snapshot max

## Key Constraints

- Do NOT modify preset-builder.ts or podgo-builder.ts
- Stadium in v3.0 is single-path only (Path 1A) — no dual-amp, no multi-path
- STADIUM_MAX_BLOCKS_PER_PATH = 12 (from STADIUM_CONFIG in config.ts)
- Stadium mandatory EQ: "Stadium Parametric EQ" (HD2_EQParametric7Band)
- getAllModels() must be updated to include STADIUM_AMPS + STADIUM_EQ_MODELS so validate.ts VALID_IDS set picks them up

## Implementation Plan

### Task 1: Update getAllModels() in models.ts

Add STADIUM_AMPS and STADIUM_EQ_MODELS to the spread so validate.ts automatically includes Agoura_ IDs in VALID_IDS.

### Task 2: Update chain-rules.ts

1. Import `isStadium`, `STADIUM_AMPS`, `STADIUM_EQ_MODELS` from models, `STADIUM_CONFIG` from config
2. In `assembleSignalChain()`:
   - Detect stadium device
   - For amp resolution: check STADIUM_AMPS first (when stadium), then fall back to AMP_MODELS
   - For mandatory EQ: use `STADIUM_EQ_MODELS["Stadium Parametric EQ"]` when stadium
   - For block limit validation: use `STADIUM_MAX_BLOCKS_PER_PATH` (12) when stadium
   - No dual-amp for Stadium in v3.0 (isDualAmp forced false)

### Task 3: Update validate.ts

1. Import `isStadium` from types, add Stadium system model IDs (P35_* I/O)
2. In `validatePresetSpec()`:
   - Add Stadium branch: max 12 blocks total (not per-DSP), max 8 snapshots
   - Stadium uses single DSP (dsp0) — all blocks must be dsp0

### Task 4: Run tests + build

`npx vitest run` must pass 108/108 (plus any new Stadium tests).
`npm run build` must pass cleanly.
