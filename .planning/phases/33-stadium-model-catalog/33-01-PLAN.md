# Phase 33 Plan — Stadium Model Catalog

**Phase:** 33
**Plan:** 01

## Objective

Add a Stadium-specific model catalog to models.ts:
- `stadiumOnly?: boolean` flag on HelixModel interface
- `STADIUM_AMPS` with at least 10 `Agoura_*` entries
- `STADIUM_EQ_MODELS` with the Stadium 7-band Parametric EQ

## Stadium EQ

Stadium-specific 7-band Parametric EQ (replaces Helix 5-band Parametric EQ):
- ID: `HD2_EQParametric7Band` (follows HD2_ convention for Stadium effects)
- Note: Simple EQ (`HD2_EQSimple3Band`), Low/High Cut (`HD2_EQLowCutHighCut`) are NOT available on Stadium

## Implementation Plan

### Task 1: Extend HelixModel interface + add stadiumOnly flag

Add `stadiumOnly?: boolean` to HelixModel interface in models.ts.

### Task 2: Add STADIUM_AMPS constant
