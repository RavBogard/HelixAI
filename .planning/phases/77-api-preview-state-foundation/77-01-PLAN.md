    State shape:
    - `device: DeviceTarget` (default: "helix_lt")
    - `baseBlocks: BlockSpec[]` (default: [])
    - `snapshots: SnapshotSpec[]` (default: 4 empty snapshots — each with empty blockStates and parameterOverrides)

    - `selectedBlockId: string | null` (default: null)

    Actions (all use `set()` from Zustand):
    - `hydrate(device, baseBlocks, snapshots)`: Replace ALL state fields. Reset activeSnapshotIndex to 0, selectedBlockId to null. This is called when /api/preview returns.
    - `setActiveSnapshot(index)`: Clamp index to 0-3 range, set activeSnapshotIndex.
    - `selectBlock(blockId | null)`: Set selectedBlockId.
    - `setParameterValue(blockId, paramKey, value)`: Write to `snapshots[activeSnapshotIndex].parameterOverrides[blockId][paramKey]`. Create the nested objects if they don't exist. NEVER modify baseBlocks.
    - `moveBlock(blockId, newPosition)`: Find block in baseBlocks by matching blockId pattern, update its dsp, position, path fields. Return `{ success: boolean; error?: string }`.
    - `swapBlockModel(blockId, newModelId)`: Find block in baseBlocks, update modelId to newModelId, update modelName to newModelId (downstream phases will resolve display name), reset parameters to `{}`.

    Computed selectors (standalone functions, NOT in store — Zustand pattern):
    ```typescript
    export function getEffectiveBlockState(state: StoreState, blockId: string) {
      const block = state.baseBlocks.find(b => generateBlockId(b) === blockId);
      if (!block) return null;
      const snapshot = state.snapshots[state.activeSnapshotIndex];
      const overrides = snapshot?.parameterOverrides?.[blockId] ?? {};
      const bypassed = snapshot?.blockStates?.[blockId] ?? block.enabled;
      return {
        ...block,
        parameters: { ...block.parameters, ...overrides },
        enabled: bypassed,
      };
    }

    export function getBlocksByDsp(state: StoreState) {
      const dsp0 = state.baseBlocks.filter(b => b.dsp === 0).sort((a, b) => a.position - b.position);
      const dsp1 = state.baseBlocks.filter(b => b.dsp === 1).sort((a, b) => a.position - b.position);
      return { dsp0, dsp1 };
    }
    ```

    Block ID generation helper (internal, also exported for use by API hydration):
    ```typescript
    export function generateBlockId(block: BlockSpec, index?: number): string {
      // Use type + position as stable ID: "amp0", "delay2", "cab0"
      return `${block.type}${block.position}`;
    }
    ```
