<objective>
## Goal
Add bass amp and cab model data to the model database, add an `instrument` field to ToneIntent so the AI can distinguish guitar vs bass presets, and update all family catalogs to include bass model names.

## Purpose
Bass guitarists currently cannot generate presets — the system has no bass amp/cab models and no way to express "bass" as an instrument type. This plan lays the data foundation for full bass support.

## Output
- 19 bass amp models added to AMP_MODELS with bass-appropriate defaultParams
- 8 bass cab models added to CAB_MODELS with bass-appropriate defaultParams
- `instrument` field ("guitar" | "bass") added to ToneIntent schema
- All 4 family catalogs updated with bass model names
</objective>
