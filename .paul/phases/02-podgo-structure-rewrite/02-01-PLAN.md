<objective>
## Goal
Rewrite the Pod Go builder to match exact structural patterns from real Pod Go Edit .pgp exports. Fix the three critical constant errors (snapshot controller ID, footswitch indices, block count), ensure snapshots include all blocks including cabs, and pad DSP to always emit exactly 10 blocks (block0-block9).

## Purpose
Current Pod Go presets have wrong snapshot controller ID (@controller: 4, should be 11), wrong footswitch indices (0-5, should be 1-6), variable block counts (should always be 10), and snapshots that skip cab blocks. These structural mismatches mean presets may not load correctly in Pod Go Edit or behave unexpectedly on hardware.

## Output
- types.ts: POD_GO_SNAPSHOT_CONTROLLER fixed 4 → 11, POD_GO_STOMP_FS_INDICES fixed [0-5] → [1-6]
- podgo-builder.ts: DSP always emits 10 blocks, snapshots include all blocks (including cabs), empty snapshots have full structure, blockKeyMap simplified (no cab-skip)
- Tests updated for new constants and structure
</objective>
