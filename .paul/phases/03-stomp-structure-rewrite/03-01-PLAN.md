<objective>
## Goal
Rewrite the HX Stomp builder to match exact structural patterns from real HX Stomp .hlx exports. Fix missing I/O (inputB/outputB), missing split/join, wrong snapshot controller ID (19 → 9), wrong footswitch indices (0-based → 1-based), wrong @pedalstate (2 → 0), missing global fields, and hardcoded block type resolution.

## Purpose
Current HX Stomp presets are missing structural elements that every real preset has (inputB, outputB, split, join) and use wrong constant values (snapshot controller, FS indices, pedalstate). These mismatches mean presets may not load correctly in HX Edit or behave unexpectedly on hardware.

## Output
- types.ts: New STOMP_SNAPSHOT_CONTROLLER = 9, STOMP_FS_INDICES/STOMP_XL_FS_INDICES updated
- stomp-builder.ts: inputB/outputB added, split/join added, correct controller/FS/pedalstate, device-specific block types, complete global section
</objective>
