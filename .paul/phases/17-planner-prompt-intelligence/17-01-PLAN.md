## Goal
Strengthen the planner prompts so the AI makes smarter creative decisions: better effect combinations, more accurate snapshot role guidance, and stronger device-specific best practices. Reduce the gap between what the planner generates and what a professional preset creator would choose.

The audit shows all families pass intent fidelity and musical intelligence, but the planner's creative choices can be improved. Better prompt intelligence means better presets without touching the builder or Knowledge Layer.

## AC-2: Snapshot Role Guidance in Planner Prompt
```gherkin
Given a planner prompt for any device family
When the prompt's creative guidelines section is read
Then it includes explicit guidance on what each toneRole means for the preset character:
  - clean: bypasses drive/distortion, minimal effects, amp sparkle
  - crunch: moderate drive engaged, rhythm-focused
  - lead: drive + boost engaged, effects for sustain
  - ambient: time-based effects fully engaged, drive minimal
And the planner understands HOW snapshot bypass toggling creates the variation
```

## AC-3: Amp Selection Intelligence
```gherkin
Given a planner prompt for any device family
When the prompt's amp-cab pairing section is read

  - Clean/edge-of-breakup amps (Fender, Vox)
  - Medium-gain amps (Marshall Plexi, Matchless)
  - High-gain amps (Mesa Rectifier, Friedman, 5150)
And the planner can match amp gain level to the user's described tone
```

## AC-4: Effect Role Assignment Intelligence
```gherkin
Given a planner prompt for any device family
When the prompt's effect intelligence section is read
Then it includes guidance on when to use each effect role:
  - always_on: core tone-shaping (compressor, EQ, always-on reverb)
  - toggleable: switched per snapshot (drive, boost, wah)
  - ambient: pads and textures (shimmer reverb, swell delays)
And the guidance prevents common mistakes (e.g., marking drive as "ambient" or reverb as "toggleable" when it should be always_on)
```
