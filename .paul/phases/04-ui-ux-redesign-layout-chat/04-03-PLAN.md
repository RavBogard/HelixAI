<objective>
## Goal
Redesign PresetCard, SignalChainViz, ToneDescriptionCard, and SubstitutionCard with improved visual hierarchy, scannability, and polish — matching the chat typography and styling patterns established in Plans 01-02.

## Purpose
PresetCard is the culmination of the user's conversation — the final deliverable. It must look premium and be easy to scan. Currently uses inline Tailwind with inconsistent sizing relative to the chat redesign. Bringing it in line with the chat polish completes Phase 4's visual redesign scope.

## Output
- Restyled PresetCard.tsx with CSS-driven styling (globals.css classes)
- Improved SignalChainViz block readability
- Better snapshot badge and effect tag visual hierarchy
- SubstitutionCard polish for confidence-level visual clarity
</objective>
