<objective>
## Goal
Enhance the builder logic so snapshots produce musically distinct tones — not just block on/off toggling. Add per-snapshot effect parameter overrides (reverb Mix, delay Mix) and genre-modulated snapshot tuning. Align builder behavior with the Phase 17 planner prompt's snapshot role table.

## Purpose
Phase 17 taught the planner to promise distinct snapshot behavior (clean=subtle reverb, lead=lush reverb+delay, ambient=long tails). The builder currently only toggles blocks on/off per snapshot — it doesn't adjust effect parameters per role. This means a "clean" snapshot and a "lead" snapshot both get the same reverb Mix when reverb is enabled. This phase closes that gap so presets actually deliver what the planner promised.

## Output
- Enhanced snapshot-engine.ts with per-role effect parameter overrides for reverb and delay
- Genre-modulated snapshot tuning (metal vs ambient vs blues lead snapshots sound different)
- Combination adjustment for drive+reverb (reduce reverb when drive is engaged)
- All existing tests pass + new tests covering the enhancements
</objective>
