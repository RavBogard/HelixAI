<objective>
## Goal
Add Helix Native as a fully supported device target. Users can select "Helix Native" in the device picker, generate presets through the AI planner, and download valid .hlx preset files.

## Purpose
Helix Native is Line 6's software-only plugin version of the Helix hardware. It uses the same HD2 amp/cab/effect models and the same .hlx file format as Helix Floor/LT/Rack, making it a natural addition to the supported device lineup. It's a popular product among recording guitarists who don't need hardware.

## Output
- `helix_native` added as a DeviceTarget
- Device picker shows "Helix Native" option
- Full preset generation flow works end-to-end (planner → knowledge layer → .hlx download)
</objective>

## Architecture Notes
- Helix Native resolves to `family: "helix"` — uses HELIX_AMP_NAMES, HELIX_CAB_NAMES, HELIX_EFFECT_NAMES catalogs
- Same .hlx file format as Floor/LT/Rack — uses buildHlxFile() in preset-builder.ts
- Same dual-DSP architecture, same capabilities as Helix Floor
- Key difference from hardware: NO Variax support (no VDI jack — it's a DAW plugin)
- Expression pedals: Still supports expression control via MIDI/automation, so snapshot design is valid
- Device ID: Helix Native uses the same .hlx format; device ID 2162690 (0x210002) based on Line 6 firmware conventions (Floor=0x210001, LT=0x210004, Rack=0x210001, Native likely 0x210002). Mark as UNVERIFIED.
